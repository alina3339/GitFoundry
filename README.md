# GitFoundry

A precision-built website for a UK-based web design studio. This repository contains the core source code for [www.gitfoundry.co.uk](https://www.gitfoundry.co.uk).

GitFoundry builds websites for local businesses, independent professionals, researchers, and charities — one-time payment, hosted free on GitHub Pages, full source code handed over on completion.

---

## The approach
Every site is hand-coded from scratch — no templates, no CMS bloat. Fast on mobile, accessible to all users, structured for search engines, and owned outright by the client from day one.

## 🛠 Technical Specifications
* **Architecture:** Static HTML5 / CSS3 / Vanilla JavaScript.
* **Performance:** Optimized for 100/100 Lighthouse scores.
* **Design System:** Custom-built grid-based CSS, zero external frameworks.
* **Deployment:** Automated CI/CD via GitHub Actions to a custom `.co.uk` endpoint.

## 📂 Project Structure
* `index.html` — The flagship landing experience.
* `portfolio.html` — Curated project showcase.
* `pricing.html` — Tiered service architecture.
* `brief.html` — Project onboarding and requirement gathering.
* `legal.html` — Regulatory compliance and privacy documentation.
