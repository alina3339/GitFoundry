Services

What we build

Most platforms rent you a website. We build you one you actually own — hand-coded, designed for your goals, and hosted free on GitHub Pages with no expiry date.

The core deliverable

A hand-coded website
you permanently own.

Design
Elegant layouts, not templates We design around your goals — audience, tone, and purpose. Every layout decision is intentional. Nothing is dragged-and-dropped.
Engineering
Semantic structure, lean styling, minimal scripts Built without templates. Accessible, fast, and optimised for search engines. No CMS bloat, no security vulnerabilities from unused plugins.
Accessibility
WCAG 2.1 AA as standard Proper heading structure, keyboard navigation, screen reader support, and sufficient colour contrast — built in from the start.
Performance
Lighthouse 90+ across all metrics No unnecessary scripts, no render-blocking fonts, no oversized images. Performance is a design constraint, not an afterthought.
Deployment
GitHub Pages — free, indefinitely Your site stays live as long as GitHub exists. No renewal dates, no hosting invoices, no platform risk.
Ownership
Full source code handover You receive the complete repository. Every file, every commit. You can take it anywhere, give it to another developer, or edit it yourself.
How it works

From brief to live
in under two weeks.

You deal with the same person throughout — the one designing and building your site. No handoffs, no agency layers.

01
Brief

You complete our short project brief — goals, audience, tone, any reference sites. No calls required unless you prefer them.

02
Design & build

We design and build your site. You receive a preview link, typically within five working days.

03
Revisions

You review and request changes. Each package includes a defined number of revision rounds.

04
Handover

We deploy to GitHub Pages, connect your domain, and hand over the full source repository. The site is yours.

Scope & boundaries

What GitHub Pages supports — and what it doesn't.

GitHub Pages serves static files. This covers the vast majority of professional website needs.

Supported
Local business sites · Portfolio sites · CV & résumé pages · Business landing pages · Research publications · Documentation · Event pages · Non-profit homepages · Professional profiles · Writer sites · Contact forms (via Formspree)
Not supported
Server-side databases · User login systems · E-commerce checkout · Real-time feeds · User-generated content If your project needs any of these, we'll discuss the right approach before quoting.
